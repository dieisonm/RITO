# Playbooks da Fase 1

Esta pasta reúne os playbooks operacionais da fase 1 da equipe nativa de agentes da RITO.

## Agentes cobertos

- atendimento e relacionamento
- comercial
- conteúdo orgânico
- creative production / social design ops
- growth / aquisição
- prospecção e inteligência comercial
- disparo / outbound
- revisor

## Como usar

Cada playbook descreve o fluxo prático do agente, os gatilhos, os critérios de qualidade, o que registrar na memória e quando escalar para humano.

## Regra de operação

- trabalhar com linguagem clara e profissional
- focar em micro e pequenas empresas
- evitar buzzwords e promessas vagas
- registrar aprendizados relevantes na memória persistente
- escalar qualquer decisão sensível para revisão humana
