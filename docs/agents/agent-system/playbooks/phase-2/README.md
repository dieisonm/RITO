# Fase 2 - Playbooks

Esta pasta reúne os playbooks operacionais da Fase 2 da operação nativa de agentes da RITO Sistemas.

## Cobertura

- Branding
- Site / UX / Conversão
- Operações / Delivery
- Financeiro / Pricing

## Como usar

Cada playbook deve orientar o agente especialista em:

- objetivo do trabalho
- gatilhos de acionamento
- fluxo passo a passo
- critérios de qualidade
- checklist de saída
- registro de memória
- critérios de escalonamento para humano

## Fluxo recomendado

1. identificar a demanda
2. selecionar o playbook correspondente
3. executar o fluxo passo a passo
4. validar a qualidade da saída
5. registrar a decisão ou aprendizado na memória
6. escalar quando houver risco comercial, institucional, operacional ou financeiro

## Observação

Os playbooks desta fase devem ser usados em conjunto com os prompts de cada agente e com a memória persistente do projeto.
