# Playbooks dos Agentes

Esta pasta organiza os playbooks operacionais da equipe-base da RITO por fase de implantação.

## Estrutura

- `phase-1/`: agentes que entram primeiro em operação
- `phase-2/`: agentes complementares que refinam a operação

## Como usar

1. identificar o agente dono da demanda
2. abrir o playbook da fase correspondente
3. seguir fluxo, checklist, gatilhos de memória e escalonamento
4. registrar a saída na pasta operacional correta

## Observação

Os playbooks trabalham junto com:

- `prompts/`
- `memory-protocol.md`
- `handoff-rules.md`
- `approval-matrix.md`
