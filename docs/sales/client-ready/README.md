# Client-Ready Templates

Esta pasta concentra os documentos prontos para uso real com cliente na RITO Sistemas.

## Objetivo

Transformar a base comercial e operacional em materiais que possam ser usados no atendimento diario, na criacao de propostas e no envio de acompanhamento sem precisar reescrever tudo do zero.

## Como usar

1. Comece por `first-contact-playbook.md` para padronizar a primeira resposta.
2. Use `client-briefing-form-final.md` para coletar informacoes com o cliente.
3. Use `estimate-final-template.md` para montar a estimativa inicial.
4. Use `commercial-proposal-final-template.md` para enviar a proposta formal.
5. Antes de enviar qualquer documento, confira `proposal-send-checklist.md`.
6. Depois do envio, siga `follow-up-sequence-final.md` para manter a conversa organizada.

## Principios

- Linguagem clara, profissional e simples.
- Foco em micro e pequenas empresas.
- Orcamento gratis e sem compromisso, quando aplicavel.
- Atendimento proximo, sem exagero tecnico.
- Documentos prontos para uso operacional, nao apenas conceitual.

## Observacoes

- Estes textos podem ser adaptados por projeto, mas a estrutura base deve ser preservada.
- Sempre que o cliente trouxer um caso novo, o briefing deve ser preenchido antes da proposta.
- Se houver mudanca de escopo, refaca a estimativa e ajuste a proposta antes de reenviar.
