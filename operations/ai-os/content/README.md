# Content

Esta pasta organiza a operacao de conteudo da RITO.

## Blocos recomendados

- pautas
- rascunhos
- aprovados
- publicados
- reaproveitamento

## Regra editorial

Todo conteudo deve:

- partir de uma dor real de pequenas empresas
- explicar de forma simples
- mostrar utilidade pratica
- reforcar a proposta da RITO sem parecer propaganda vazia
