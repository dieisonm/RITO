# Operations

Esta pasta reune as frentes operacionais da RITO Sistemas.

## Estrutura atual

- `website/`: evolucao do site institucional
- `linkedin/`: estrategia e operacao de LinkedIn
- `instagram/`: avaliacao e operacao de Instagram
- `content-studio/`: producao e reaproveitamento de conteudo
- `ai-os/`: sistema operacional para organizar o trabalho de agentes e automacoes

## Regra geral

Cada frente deve ter:

- um objetivo claro
- materiais de entrada
- entregaveis padronizados
- historico minimo do que foi decidido

## Integracao com a base documental

Os documentos mestres ficam em [`docs/company/`](../docs/company/README.md). A pasta `operations/` existe para organizar o trabalho em andamento e a execucao pratica.
