## Details

Foram atualizados os arquivos de agentes para incluir: declaração obrigatória de formato (estático, carrossel ou reel), regra image-first para posts estáticos, limite de texto na arte, uma ideia por slide em carrosséis, verificação de impacto em thumbnail e pergunta binária de revisão: isso parece um post nativo do Instagram? O branding agora também precisa bloquear estruturas com cara de deck ou flyer institucional.
