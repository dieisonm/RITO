## Details

A atualização padronizou a exibição do número para o contexto brasileiro e adotou uma convenção mais comum em sites institucionais: número visível e clicável, com ícones discretos para canais digitais. Como o repositório ainda não contém URLs oficiais de Instagram e LinkedIn, esses dois ícones foram renderizados como elementos visuais de 'em breve' em vez de links inventados. O WhatsApp permanece como link ativo com mensagem inicial preenchida.
