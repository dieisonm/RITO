## Details

Brand decisions should anchor on docs/brand-foundation.md and the actual site implementation. The official palette includes warm off-white backgrounds, dark petroleum blue, graphite ink and sand accent. Official typography is Sora for titles, Manrope for body/interface and IBM Plex Mono for structural labels. The institutional brand guide and software UI brand system translate that into practical usage rules.
