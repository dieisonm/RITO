## Details

The pricing workflow now combines stage-based hours, service type reference range, package layer (Essencial, Operacional, Sob Medida), scenario layer (Mínimo defensável, Ideal comercial, Premium estratégico), installment simulation, proposal summary export and execution history. The pricing guide and calculator guide document how the team should use it.
