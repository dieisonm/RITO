# Ops

Esta pasta reúne o trabalho operacional vivo da RITO.

## Estrutura

- `ai-os/`: operação central com WhatsApp, growth, propostas, suporte e templates.
- `instagram/`, `linkedin/`, `meta-ads/`, `website/`: frentes específicas de canal.
- `content-studio/`: produção e reaproveitamento de conteúdo.

## Regra geral

Cada frente deve ter:

- um objetivo claro
- materiais de entrada
- entregaveis padronizados
- historico minimo do que foi decidido
