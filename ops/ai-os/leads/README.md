# Leads

Esta pasta guarda leads ja entendidos minimamente.

## Para cada lead registrar

- nome do contato
- empresa ou atividade
- cidade e contexto
- problema principal
- urgencia percebida
- canal preferido
- proximo passo comercial

## Objetivo

Permitir que o agente comercial retome rapidamente o contexto e monte proposta sem recomecar do zero.
