# Playbooks

Esta pasta guarda rotinas padronizadas para os agentes da RITO.

## Objetivo

Documentar o passo a passo das operacoes recorrentes para reduzir improviso e manter consistencia.

## Relacao com a base canonica

Os playbooks canonicos da arquitetura ficam em `docs/agents/agent-system/playbooks/`.

Esta pasta de `ops/ai-os/playbooks/` deve ser usada para:

- versoes operacionalizadas de uso frequente
- checklists de campo
- adaptacoes por rotina real de atendimento, comercial, growth ou delivery
- materiais vivos que nascerem da operacao do dia a dia

## Quando usar

Use esta pasta para rotinas como:

- triagem de inbox
- qualificacao de lead
- montagem de proposta
- resposta de atendimento
- briefing de campanha
- revisao final
- onboarding de cliente

## Convencao de Nomes

Use nome da rotina com sufixo `playbook`.

Exemplos:

- `lead-qualification-playbook.md`
- `proposal-building-playbook.md`
- `whatsapp-response-playbook.md`
- `content-review-playbook.md`
