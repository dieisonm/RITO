# Templates

Esta pasta guarda modelos prontos para copiar, preencher e registrar em cada frente operacional.

## Objetivo

Padronizar a entrada de informacao, o estado da demanda, os proximos passos e o registro de memoria.

## Quando usar

Use esta pasta sempre que precisar criar:

- dossies
- briefings
- propostas
- logs
- revisoes
- campanhas
- registros de entrega

## Convencao de Nomes

Use o nome da frente com sufixo `template`.

Exemplos:

- `lead-dossier-template.md`
- `proposal-package-template.md`
- `content-brief-template.md`
- `review-note-template.md`

## Quem Atua
