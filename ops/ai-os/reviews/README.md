# Reviews

Esta pasta guarda revisoes criticas, riscos e decisoes de aprovacao.

## Objetivo

Manter um ponto unico para registrar o que foi aprovado, o que precisa de ajuste e o que nao deve seguir adiante.

## Quando usar

Use esta pasta quando houver:

- revisao de proposta
- revisao de contrato
- revisao de post, campanha ou texto
- revisao de pagina do site
- revisao de resposta de atendimento
- decisao de bloquear ou aprovar uma entrega

## Convencao de Nomes

Use data, item revisado e versao.

Exemplos:

- `2026-04-14-review-proposta-v1.md`
- `2026-04-14-review-homepage-v2.md`
- `2026-04-14-review-atendimento-whatsapp.md`

## Quem Atua

- `Revisor`
- `Orquestrador`, quando precisar consolidar apontamentos
- `Comercial`, `Conteudo`, `Growth` ou `Atendimento`, quando houver retorno para ajuste
