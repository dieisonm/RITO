# Instagram Stories - Codex ImageGen Test

## Objetivo

Registrar o primeiro teste operacional de geração de imagem para Story da RITO feito diretamente no Codex com ImageGen.

## Decisão

A partir deste teste, a RITO passa a poder gerar imagens de campanhas, stories, posts e mockups diretamente no Codex, sem depender de uma ferramenta externa, quando o usuário pedir produção visual.

## Prompt usado

```text
Criar uma imagem final pronta para Instagram Stories da RITO Sistemas, formato vertical 9:16, dimensão desejada 1080x1920.

Uso:
- Plataforma: Instagram Stories.
- Campanha: Projeto Piloto RITO para MEIs e pequenos negócios.
- Objetivo: atrair donos de pequenos negócios que têm uma rotina manual para organizar com software sob medida, planilha inteligente, automação, dashboard ou site simples.
- A imagem deve ser uma peça final completa, já com todos os textos e elementos visuais, não um mockup em branco.

Cena principal:
- Fotorealismo premium com estética editorial.
- Uma pequena empreendedora brasileira em um ateliê/loja pequena, sorrindo de forma natural, segurando um tablet com um dashboard simples e limpo.
- Ao redor, mesa de madeira com caderno, etiquetas, algumas embalagens, caneta, anotações e um celular, sugerindo rotina real de pedidos/clientes/organização.
- Luz natural quente, ambiente acolhedor, profissional e realista, sem parecer banco de imagem frio.
- A cena deve transmitir organização, proximidade, tecnologia prática e pequenos negócios.

Composição:
- Story vertical com área principal visual à direita/centro e uma faixa/overlay elegante à esquerda para texto.
- Fundo com gradiente azul petróleo profundo no lado esquerdo, integrando suavemente com a foto no lado direito.
- Safe area ampla no topo e rodapé, sem texto encostado nas bordas.
- Hierarquia clara: logo no topo, headline grande, subtítulo curto, selo/benefício, linha de soluções, CTA grande no final.
- Usar paleta RITO: azul petróleo escuro, off-white quente, dourado discreto.
- Tipografia elegante, legível e profissional, com aparência premium e leitura rápida em tela pequena.

Texto na imagem, EXATO, sem alterar e sem adicionar outros textos:
Topo / logo: "RITO SISTEMAS"
Headline principal: "Tem uma rotina que você queria organizar?"
Subtítulo: "Projeto piloto para pequenos negócios"
Selo: "Condição especial para primeiros cases"
Linha de soluções: "Controle • Automação • Planilha inteligente • Dashboard • Site simples"
CTA: "Enviar minha ideia"
Rodapé: "ritosistemas.com"

Identidade RITO:
- O texto "RITO SISTEMAS" deve aparecer como wordmark simples e legível no topo, em off-white, sem inventar letras extras.
- Se usar símbolo, usar apenas um monograma R geométrico simples ao lado do texto, sem redesenhar de forma complexa.
- Visual sóbrio, moderno, confiável, próximo e profissional.

Restrições:
- Não adicionar textos além dos especificados.
- Não cortar nenhuma palavra.
- Não encostar texto nas bordas.
- Não usar marca d'água.
- Não parecer slide, flyer, apresentação ou template corporativo genérico.
- Não usar estética futurista, neon, excesso de brilho ou interface impossível.
- Não criar logo distorcido, letras erradas ou palavra inventada.
- Não deixar o dashboard ilegível ou poluído; ele deve parecer simples e plausível.

Entrega:
- Gerar exatamente 1 imagem final.
- A imagem precisa parecer pronta para ser postada como Story orgânico da RITO, com CTA claro e visual profissional.
```

## Observações

- O usuário observou que o resultado ficou parecido com imagem anterior, mas validou que o fluxo pode ser usado.
- Próximas gerações devem pedir variação real de cena, arquétipo de negócio ou composição para evitar repetição visual.
- O ImageGen pode gerar na proporção correta, mas em resolução diferente. A etapa de produção deve validar e exportar a versão final no tamanho certo antes de considerar a peça pronta.
