# Assets

Esta pasta organiza os assets finais e os pacotes leves que ainda precisam existir no repositório.

- `business-kit/`: arquivos comerciais finais.
- `social/`: imagens, lotes, templates e fontes de peças sociais.
- `brand/`: base canônica de marca fora do runtime público do site.
- `drive/`: manifesto dos binários grandes replicados no Google Drive.

Regra:

- binários e mídias pesadas vivem no Google Drive
- o Git mantém apenas manifesto, READMEs e arquivos leves de contexto
- quando um asset novo nascer aqui, ele deve entrar no `assets/drive/asset-manifest.json`
